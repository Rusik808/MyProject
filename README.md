# Анализ данных футболистов

## Описание
Проект направлен на анализ статистики игроков и предсказание их рыночной стоимости.

## Структура проекта
- `data/`: Исходные данные (all_players.csv)
- `notebooks/`: Jupyter ноутбуки с анализом
- `src/`: Скрипты для обработки данных

## Как запустить
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите ноутбук в папке `notebooks/`.
